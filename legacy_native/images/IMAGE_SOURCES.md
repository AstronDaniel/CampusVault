## Image Optimization

Before adding images to the repository:
1. Optimize file size using tools like:
   - TinyPNG (https://tinypng.com)
   - ImageOptim (https://imageoptim.com)
   - Squoosh (https://squoosh.app)

2. Target file sizes:
   - Banner: < 500KB
   - Screenshots: < 200KB each
   - Icons: < 50KB each
